# 局域网同步音乐

在同一局域网里，一台设备上传音频并控制播放，其他设备打开同一个网站地址即可同步播放。

## 启动

```powershell
python app.py
```

启动后终端会显示：

```text
本机: http://localhost:3000
局域网: http://你的局域网IP:3000
```

把“局域网”地址发给同一 Wi-Fi 下的其他设备即可。

## 使用

1. 主持设备打开网站，上传音频。
2. 其他设备打开同一个局域网地址，只点“启用声音”。
3. 主持设备点播放、暂停或拖动进度，其他设备会跟随同步。

提示：手机和部分浏览器必须先点一次“启用声音”，这是浏览器限制。不要在手机上点播放来当主控，除非你重新上传音频。
